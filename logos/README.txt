PON AQUÍ LOS LOGOS EN ESTAS RUTAS EXACTAS:
(Los nombres deben coincidir exactamente)

  dashboard_logo.svg      → Logo principal del dashboard (cabecera)
  miniapp_logo.svg        → Logo de la mini-app de Telegram

  p1_renovacion.svg       → Renovación Popular
  p2_fuerza.svg           → Fuerza Popular
  p3_ahora.svg            → Ahora Nación
  p4_podemos.svg          → Podemos Perú
  p5_alianza.svg          → Alianza Por el Progreso
  p6_buen_gobierno.svg    → Partido Buen Gobierno
  p7_pais.svg             → País para Todos

FORMATOS ACEPTADOS:
  • SVG  → IDEAL  (escala sin perder calidad, tamaño mínimo)
  • PNG  → OK     (fondo transparente, máx 15 KB)
  • WebP → OK     (mejor compresión que PNG)
  • JPG  → NO recomendado (sin transparencia)

TAMAÑO RECOMENDADO:
  • logos partidos   → 64×64 px (se redimensionan a 28 px)
  • dashboard_logo   → 40×40 px
  • miniapp_logo     → 64×64 px

CAMBIAR EL PARTIDO CLIENTE DEL SISTEMA:
  Si mañana el adquiridor es otro partido,
  edita logos.config.js y cambia SOLO esta línea:

      clienteKey: 'p1',   ← pon 'p2', 'p3', etc.

  Todo lo demás (ranking, zona destacada, color principal)
  se actualizará automáticamente.

SI NO TIENES UN LOGO TODAVÍA:
  Puedes pasarlos en el chat y Antigravity los agrega.
  Mientras, el sistema muestra la inicial del partido
  como fallback automático (sin errores).
